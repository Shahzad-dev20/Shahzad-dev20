## 🚀 Featured Projects

<details>
<summary><b>🚗 AutoDeals.pk Car Price Intelligence Pipeline</b></summary>
<br/>

End-to-end pipeline that scrapes live vehicle listings from AutoDeals.pk and transforms raw, messy automotive data into a leakage-safe, production-grade dataset ready for predictive pricing models.

| Category | Details |
|---|---|
| **Stack** | Python, Selenium, BeautifulSoup, Pandas, Scikit-learn |
| **Scale** | Multi-page dynamic scraping with pagination handling |
| **Performance** | Leakage-corrected preprocessing (outlier bounds computed post-split) |
| **Security** | Rate-limited, polite scraping with retry/backoff logic |
| **Impact** | Reusable "fit on train, transform on both" pipeline pattern |
| **Repository** | [View Repository](https://github.com/Shahzad-dev20) |

This project was a deep dive into real-world data engineering discipline. The key breakthrough was identifying and correcting a subtle **data leakage issue** — outlier bounds were originally computed before the train/test split, silently inflating validation performance. Fixing this instilled a rigorous, universal rule now applied across every scaling, encoding, and imputation step in subsequent work.

</details>
